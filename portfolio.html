About Me

Hello my name is Derek Chow and I am the creator of DCCD productions which is a website I made to showcase my projects over the years. Currently I am a student at University of California, Irvine and work on these projects whenever I have some free time. Some of these projects have other collaborators who helped me create them, but each of them help me practice and perfect my programming skills on a higher level. My main goal as a developer is to create projects that help me grow and shape me to become a better programmer. I am a versatile programmer, and have learned many different programming skills. From high-level object oriented programming languages like C++, Java, and Python to Front-end development like HTML, CSS, and Javascript or even some game engines using Unity3d with C#. I strive to continue learning from every experience I go through and hope that one day I will be prepared for anything I have to implement in the near future.

Twitch Streamer TV

Project Details

  • A web application that contains a listing similar to a TV listing
  • Created through HTML, CSS, Javascript, and Jquery
  • Used Twitch.Tv's api to obtain the status of the list of streamers
  • Keeps track of their online/offline status and what they were doing if they are online

  • This was one of the mini-projects assigned by FreeCodeCamp for Front-End Certification
