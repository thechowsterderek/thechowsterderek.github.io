FabFlix

Fabflix is a dynamic web project, to create a fake e-commerce website which is connected to a mysql database to keep track of stock, users, and employees. The website is also hosted through Amazon Web Services EC2 instance with tomcat and apache2. Although it requires a login to access the website, anyone can sign-up through the login page and add themselves to the exclusive group. The e-commerence website main focus is to sell movies, but also lists actors, genres, and unique identification numbers to make it more convenient to sort or find. Supports master-slave instances for a more effcient design for requests to the database.(Separates reads and writes for mySQL server).

  • Utilizes bootstrap3, HTML5, CSS3, and javascript to create a responsive website that works for all sizes
  • A mysql database that keeps track of users,movies, quantity of movies, sales, employees and much more to create an organized environment
  • Web hosted through multiple Amazon Web Service, EC2 instances with Apache2, Tomcat, and so forth to create an optimal solution for the backend problems
  • JSP files and java servlets was used to create dynamic webpages allowing easy includes and information to be easily transfer from webpage to webpage
